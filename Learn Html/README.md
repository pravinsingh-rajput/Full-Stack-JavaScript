# Learning HTML

## Heading Tag

```html
<h1>Heading</h1>
<h2>Heading</h2>
<h3>Heading</h3>
<h4>Heading</h4>
<h5>Heading</h5>
<h6>Heading</h6>
```

## Paragraph Tag

```html
<p>Lorem ipsum dolor sit amet consectetur.</p>
```

## Title Tag

```html
<title>Webpage Title</title>
```

## Style Tag

```html
<style>
  h3 {
    color: red;
  }
</style>
```

## Anchor Tag

```html
<a href="https://google.com">Google</a>
```
